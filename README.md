# Yii2 Requests Report (Yii2 + Docker + PostgreSQL)

Тестовое задание: система заявок + отчёт по **вирішеним** заявкам (resolved) в разрезе агентов и категорий, с фильтром по **дате решения**.

## Функциональность

### Заявки (requests)
Заявка содержит:
- заголовок (title)
- описание (description)
- агент (agent_id)
- статус: **Нова / В роботі / Вирішена**
- категория: **Відключення / Перевірка/здешевлення / Технічне питання / Інше**
- дата создания (created_at)
- дата решения (resolved_at, заполняется при статусе "Вирішена")

### Агенты (agents)
- справочник агентов (ФИО)

### Отчёт (report)
- фильтр по диапазону **даты решения**
- таблица: **агент × категории** (сумма по строке)
- учитываются только заявки со статусом **"Вирішена"**

## Роуты (страницы)

- `/request/index` — список заявок
- `/request/create` — создание заявки
- `/request/update?id=1` — редактирование заявки
- `/request/view?id=1` — просмотр заявки

- `/agent/index` — список агентов
- `/agent/create` — создание агента

- `/report/index` — отчёт по решенным заявкам (фильтр по resolved_at)
